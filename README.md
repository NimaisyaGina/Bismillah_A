Website ini merupakan aplikasi berbasis Django yang menampilkan biodata kelompok secara publik sehingga dapat diakses oleh siapa saja tanpa perlu login. Website ini memuat informasi utama, seperti data anggota kelompok, deskripsi proyek, serta tampilan tema website yang sedang digunakan. Website ini dibuat sebagai media informasi kelompok sekaligus sebagai implementasi tugas pengembangan web yang rapi, informatif, dan mudah digunakan.
Selain itu, website ini juga menyediakan fitur login menggunakan Google Sign-In untuk anggota yang terdaftar. Login diperlukan untuk mengakses fitur khusus, seperti mengubah tema website, sedangkan pengunjung umum hanya dapat melihat isi website tanpa melakukan perubahan. Sistem ini juga dilengkapi dengan pengaturan keamanan, seperti pembatasan akses, validasi input, CSRF protection, serta pencatatan riwayat perubahan agar website tetap aman dan terkontrol.
